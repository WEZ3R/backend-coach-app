# Commande: start-session

Lis le fichier CLAUDE.md à la racine du projet pour comprendre l'architecture et les conventions de code du backend FitFlow.

## Instructions

1. Lis le fichier `CLAUDE.md` situé à la racine du projet
2. Confirme que tu as bien lu et compris le contenu en affichant "claude.md lu"
3. Tu es maintenant prêt à travailler sur le backend FitFlow en respectant les conventions établies

## Contexte

Tu travailles sur le backend de l'application FitFlow, une plateforme de coaching fitness. Le backend utilise:
- Node.js avec ES Modules
- Express.js
- Prisma ORM avec PostgreSQL
- JWT pour l'authentification
- Architecture MVC (controllers, routes, middlewares)

Respecte les conventions de code françaises et le format de réponse standardisé (sendSuccess/sendError).
